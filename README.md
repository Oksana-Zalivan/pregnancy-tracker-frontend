# 🕊️ Leleka — Pregnancy App

A modern web application designed to support women during pregnancy.  
Built with ❤️ using Next.js and modern frontend technologies.

---

## 🚀 Features

- 👶 User-friendly interface for pregnant women
- 📱 Responsive design (mobile-first)
- 🌐 Multi-language support (planned)
- 🧾 Forms for entering personal data
- 🖼️ Image upload (avatar)
- ⚡ Fast performance with Next.js

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript  
- **Styling:** CSS Modules  
- **State Management:** (додаси, якщо буде — наприклад React Query)
